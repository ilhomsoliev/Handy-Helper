# Handy-Helper

Android App that helps with setting your goals, todo-list, notes, tracking your habits and your finances made by Jetpack Compose!


## Features

- Notes
- Goals
- Todo-List
- Finances
- Habit Tracking

## Technologies

- Jetpack Compose
- Room
- Clean Architecture
- Hilt/Dagger2
- Multi-Modules
- Accompanist
